# 🦜 Cuna Alada - E-Commerce y Gestión de Agapornis

![Estado](https://img.shields.io/badge/Estado-Producción-success)
![Versión](https://img.shields.io/badge/Versión-1.0.0-blue)

Cuna Alada es una plataforma web Full-Stack diseñada no solo para la comercialización de aves Agapornis y sus accesorios, sino con una **misión ética fundamental**: garantizar que cada ejemplar tenga una vida digna. A través de nuestro módulo de "Academia", educamos a los futuros dueños sobre los cuidados, alimentación y entorno que estas aves necesitan para prosperar.

## ✨ Características Principales

* 📚 **Academia de Cuidados:** Módulo educativo integral con guías detalladas para concientizar sobre la tenencia responsable.
* 🛍️ **Catálogo y Tienda:** Exploración de ejemplares disponibles y compra de accesorios especializados con integración a WhatsApp.
* 🎮 **Jaula Interactiva:** Mini-juego y experiencia visual dinámica para interactuar y conocer a las aves de forma lúdica.
* 🤖 **Asistente Inteligente (Quiz):** Sistema interactivo que recomienda el ave ideal basándose en el estilo de vida del usuario.
* 🔒 **Panel Administrativo Privado:** Dashboard protegido por autenticación para el control de inventario (CRUD de aves y productos) en tiempo real.

## 🚀 Tecnologías Utilizadas (MERN Stack)

* **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion (Animaciones), React Router.
* **Backend:** Node.js, Express.js.
* **Base de Datos:** MongoDB (Atlas), Mongoose.
* **Despliegue:** Render (Web Service & Static Site).
