# 🦜 Cuna Alada - Sistema de Gestión de Agapornis

Este es un sistema integral para el control de inventario, cuidados y venta de aves Agapornis. Proyecto desarrollado con un enfoque profesional dividido en 6 etapas (Sprints).

## 🚀 Tecnologías Usadas
* **Frontend:** React + Vite, Tailwind CSS.
* **Backend:** Node.js, Express.
* **Base de Datos:** MongoDB.

## 📋 Resumen de Desarrollo (Sprints)
* **Sprint 1:** Armé la base del servidor con Node y Express para los cimientos del sistema.
* **Sprint 2:** Creé las fichas (modelos) para definir qué datos guardar de cada ave (nombre, precio, etc.).
* **Sprint 3:** Preparé el taller del Front configurando React, Vite y Tailwind.
* **Sprint 4:** Hice las piezas LEGO (componentes) como la `AveCard` y el botón de WhatsApp.
* **Sprint 5:** Conecté todas las páginas (Admin, Tienda, Login) para que la navegación funcione perfecta.
* **Sprint 6:** Puse la fachada final con todos los logos, fotos y el icono de la pestaña.
